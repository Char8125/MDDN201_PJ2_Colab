About

Colab is website that works similarly to crowdfunding but instead of money, it would take in your artistic contributions to a project. For example a web comic would abruptly stop cause of an artistic block, people could then "continue” the story by sending in scripts and art, to try and continue the story. It works on iterations from the community that loves the artist’s work. Colabing into a project is as simple as signing up on the website and browsing current projects, It will work on a GitHub based system integrated with a community vote system, where the community gets to up vote, down vote, report Colab submissions and Colab projects; filtering stolen content and ‘noise’.
